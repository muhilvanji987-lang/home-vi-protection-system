Home Voltage & Current Protection System

An Arduino-based safety device that continuously watches household mains voltage and appliance current, cuts power automatically before a fault becomes a short circuit, and alerts the homeowner's phone by SMS — all without any manual intervention.

01The problem

What goes wrong today

Home wiring faults — voltage surges, overloaded circuits, aging insulation — often go unnoticed until they cause a short circuit, appliance damage, or fire. Standard MCBs react only after a fault has already progressed.

What this system does differently

It watches voltage and current continuously and intervenes early — cutting appliance power the moment readings cross a safe threshold, before a short circuit can occur, and immediately telling the homeowner what happened and where.

04Core logic

// Trip the moment either reading exceeds a safe limit
bool faultNow = (voltage > VOLTAGE_LIMIT) || (current > CURRENT_LIMIT);

if (faultNow && !tripped) {
  tripped = true;
  cutPower();  // relay opens instantly
  sendSMS(reason, voltage, current);  // GSM alert
}

// Power only comes back after a manual reset AND a clear reading
if (buttonPressed && !faultNow) restorePower();

    06Outcomes

    • ✓Faster than a short circuit — the system reacts to abnormal voltage/current before the fault escalates, unlike a standard breaker that trips after the fact.
    • ✓Remote awareness — the homeowner is notified by SMS the instant a fault occurs, even if no one is home.
    • ✓Fail-safe reconnection — appliances stay off until someone manually resets the system and the fault has actually cleared, preventing repeated damage.
    • ✓Low-cost, replicable design — built entirely from commonly available modules, making it practical for real home deployment.
    Safety note — this project senses and switches real 230V mains. It was built and demonstrated under faculty supervision using isolated, certified sensor modules and an enclosed relay stage.
